A Supporting Information for “Mapping-friendly sequence reductions: going beyond homopolymer compression”

A.1 “TandemTools” dataset generation

This dataset was obtained by taking a human X chromosome HOR sequence, concatenating it 500 times with added mutations in order to obtain an approximately 1 Mbp long sequence. Then 1200 reads were simulated from the sequence using nanosim272 and assembled using a centromere-tailored pipeline529. A 10kbp deletion was then added to this assembly. The resulting sequence is the one we refer to as the “Centromeric sequence”.
